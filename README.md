# CrossRemoverProxy

CrossRemoverProxy is a lightweight proxy server designed to bypass CORS (Cross-Origin Resource Sharing) restrictions for client-side applications.

It allows frontend applications (Web, WebOS TV, React, etc.) to access external APIs that normally block cross-origin requests.

---

## 🚀 Features

- Remove CORS restrictions
- Simple and lightweight
- Easy deployment
- Supports HTTP and HTTPS
- Suitable for WebOS TV applications
- Configurable target domains

---

## 📦 Use Cases

- WebOS TV applications
- Smart TV apps
- Frontend-only projects
- Testing external APIs
- Accessing restricted resources from client-side apps
